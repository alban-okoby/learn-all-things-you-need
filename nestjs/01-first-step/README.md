### First step with nestjs

### Languages and tools
- TypeScript by default, use javasSript
- Nodejs >= 16
- Babel (for compilation)

#### Config auto restart
```
npm run start:dev
```

#### Formatting && Linting
- Linting (eslint)
```
npm run lint
```
- Formatting (prettier)
```
npm run format
```
